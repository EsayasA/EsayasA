## ⭐ Featured Projects

### 🤖 AI Recruiting Platform

A full-stack AI-powered Applicant Tracking System built with **Next.js**, **FastAPI**, **TypeScript**, **JWT authentication**, and **Ollama embeddings**.

Recruiters can create jobs, review applicants, rank candidates with AI, manage interviews, track analytics, and receive notifications. Candidates can apply for jobs, upload resumes, track status, and receive updates.

**Tech:** Next.js, React, TypeScript, FastAPI, SQLAlchemy, Ollama, Tailwind CSS

---

### 💬 RAG Chatbot

A Retrieval-Augmented Generation chatbot that lets users ask questions over a private knowledge base.

Features include document upload, semantic search, ChromaDB vector storage, Ollama-powered responses, streaming answers, conversation memory, and an admin dashboard.

**Tech:** FastAPI, Next.js, Ollama, ChromaDB, React, TypeScript

---

### 👁️ Open-Vocabulary Object Locator & Tracker

A computer vision application that combines modern foundation models for object detection, segmentation, and tracking.

Users can search for objects using natural language, detect them in images or videos, segment them, and track them across frames.

**Tech:** Python, Grounding DINO, CLIP, SAM 2, OpenCV
